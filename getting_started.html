🚀 起步

Parcel 是一个 Web 应用程序打包程序，以其开发体验而独树一帜。它利用多核处理提供极快的性能，并且是零配置的。

首先通过 Yarn 或 npm 安装 Parcel:

Yarn:

yarn global add parcel-bundler

npm:

npm install -g parcel-bundler

通过下面命令，在项目目录下创建 package.json 文件:

yarn init -y

or 

npm init -y

Parcel 可以使用任何类型的文件作为入口，但 HTML 或 JavaScript 文件是一个很好的入口。如果你使用相对路径在 HTML 中链接入口 JavaScript 文件，则 Parcel 也会为你处理，并将该引用替换为输出文件的 URL。

接下来，创建 index.html 和 index.js 文件。

<html>
<body>
  <script src="./index.js"></script>
</body>
</html>
console.log("hello world");

Parcel 有一个内置的开发服务器, 会根据你的文件改动自动重新构建，并且支持 热模块替换 进行快速开发。只需要在入口文件指出：

parcel index.html

在浏览器打开 http://localhost:1234/ 。你可以通过 -p <端口> 选项设置端口。

如果您没有自己的服务器，或者你的应用程序完全是客户端渲染的，请使用开发服务器。如果你有自己的服务器，你可以在 watch 模式下运行 Parcel。 这仍然会对文件更改自动重建，并支持热模块替换，但不启动 Web 服务器。

parcel watch index.html

当你准备建立生产时，build 模式关闭监视，并且只构建一次。有关更多详细信息，请参见生产 部分。
