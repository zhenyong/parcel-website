🚀 极速打包

Parcel 使用 worker 进程支持多核编译，并具有文件系统缓存，即使在重新启动后也可快速重建。

📦 打包所有资源

Parcel 支持 JS、CSS、HTML 以及其他文件资源 - 不需要插件。

🐠 自动转换代码

在需要的时候自动使用 Babel、PostCSS 和 PostHTML 自动转换 - 甚至可以处理 node_modules 的代码。

✂️ 零配置支持代码分块

使用动态 import() 语法，Parcel 会拆分你的代码包，于是初始化时只需加载必要的资源。

🔥 热更新

当您在开发过程中进行更改时，Parcel 会自动更新浏览器中的模块，无需进行任何配置。

🚨 友好的错误日志

遇到错误时，Parcel 会打印语法高亮的代码块，帮助你查明问题。

Hello World

从应用程序的入口 HTML 文件开始。Parcel 从它开始跟踪依赖关系，然后构建整个应用。

✏️ index.html

<html>
<body>
  <script src="./index.js"></script>
</body>
</html>

🛠 index.js

// 导入另一个组件
import main from './main';

main();

🛠 main.js

// 导入一个 CSS Module
import classes from './main.css';

export default () => {
  console.log(classes.main);
};

💅 main.css

.main {
  /* 引用一个图片文件 */
  background: url('./images/background.png');
  color: red;
}

只需要运行 parcel index.html 来启动开发服务器，接着导入 JavaScript、CSS、图片和更多文件资源，完美！👌
